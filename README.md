A complete full-stack e-commerce platform built for a Bike Showroom, allowing users to browse, buy, and sell bikes and spare parts through a secure and user-friendly web interface. The application combines a modern, responsive frontend with a scalable backend API, making it suitable for real-world commercial use. The system features secure authentication, a responsive React frontend, and a scalable Node.js + MongoDB backend, making it suitable for real-world commercial deployment.
